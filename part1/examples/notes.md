1.用Vite写出第一个React程序
2.组件 是一个函数 返回JSX(看起来像HTML标签）
3.app.jsx:负责编写组件 （交互 布局 等） main.jsx:把app.jsx的组件渲染到HTML文件里
4.也就是说，组件都将被渲染成HTML页面元素
5.严格意义上来说，一个组件相当于一个JS函数
6.不能存在一个以上的default导出，其他的用花括号命名导出
7.ES6函数定义法 箭头函数
8.react应用不显示在index.html内，显示在npm server里面 指令：npm dev run
9.让我们一起重复一遍：我保证在课程中，以及在我接下来的人生中，在开发网页时始终打开控制台。
10.组件中的html元素想要加变量：用{}花括号{a} plus {b} is {a+b}
11.在底层，由React组件返回的JSX会被编译成JavaScript。
12.可以编写多个组件，但是不用多次export，可以直接在另一个组件里以标签的形式调用其他组件
13.一个组件只能有一个return值
14.props可以向子组件传递参数，子组件通过props接收并使用这些数据。
15.props的目的就是把父组件的内容传给子组件然后渲染到浏览器标签
16.子组件的目的就是渲染到浏览器
17.组件内，声明的值用花括号
18.React组件名称的首字母必须大写
19.React组件的最外层一般要定义一个根元素（一般情况下是div）
20.React大括号里不能渲染对象，在大括号中渲染的每个东西都必须是原始值，比如数字或字符串，对象可以拆分渲染
21.层级渲染 如果只是渲染 → {} 如果是传值给组件 → name={}
